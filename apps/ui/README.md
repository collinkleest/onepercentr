# UI

This is the React UI for the onepercentr project.

## Development server

Run `nx serve ui` for a dev server. Navigate to http://localhost:4200/. The app will automatically reload if you change any of the source files.

## Build

Run `nx build ui` to build the project. The build artifacts will be stored in the `dist/apps/ui` directory.

## Linting

Run `nx lint ui` to execute the linter.

## Testing

Run `nx test ui` to execute the unit tests via [Jest](https://jestjs.io).
